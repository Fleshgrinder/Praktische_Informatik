![AVL-Baum](https://raw.github.com/Fleshgrinder/Praktische_Informatik/master/übung1/aufgabe6/avl-baum.png)

2. Fügen Sie den Knoten 11 ein (ausgehend vom Baum oben) und balanzieren Sie den Baum.
